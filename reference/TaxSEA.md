# TaxSEA: Taxon Set Enrichment Analysis

Modular TaxSEA implementation supporting enrichment (KS) and ORA
(Fisher). Provide either `taxon_ranks` for enrichment or `input_taxa`
for ORA.

## Usage

``` r
TaxSEA(
  taxon_ranks = NULL,
  input_taxa = NULL,
  mode = NULL,
  lookup_missing = FALSE,
  min_set_size = 5,
  max_set_size = 300,
  custom_db = NULL
)
```

## Arguments

- taxon_ranks:

  Named numeric vector of statistics (e.g., log2 fold changes). Required
  for enrichment.

- input_taxa:

  Character vector of taxa to treat as "hits"/selected taxa. Required
  for ORA.

- mode:

  Character. One of `"enrichment"` or `"ora"`. If NULL, inferred from
  which input is provided.

- lookup_missing:

  Logical indicating whether to fetch missing NCBI IDs. Default is
  FALSE.

- min_set_size:

  Minimum size of taxon sets to include in the analysis. Default is 5.

- max_set_size:

  Maximum size of taxon sets to include in the analysis. Default is 100.

- custom_db:

  A user-provided list of taxon sets. If NULL (default), the built-in
  database is used.

## Value

A list of data frames with taxon set results.
